# Snake

funny silly little agent to play google snake portal 3 variation, using torch.

## Requirements

Here are the versions of the main libraries used in this project:

- Python 3.10.11
- torch 2.7.1+cu118
- pygame 2.6.1
- numpy 1.26.4
- matplotlib 3.10.3
- imageio 2.37.0

## Run

To run the snake agent, execute the following command in your terminal:

```bash
python large.py
```

to show:

```bash
python large.py show
```

to make a gif:

```bash
python large.py gif
```
